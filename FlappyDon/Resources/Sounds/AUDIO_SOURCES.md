## Common License Types

**CC0 (Public Domain):**
- No attribution required
- Commercial use allowed
- No restrictions

**CC-BY 4.0 (Creative Commons Attribution):**
- Attribution required
- Commercial use allowed
- Must credit author

**CC-BY-SA 4.0 (Creative Commons Attribution-ShareAlike):**
- Attribution required
- Commercial use allowed
- Derivative works must use same license

**Work-for-Hire:**
- Full rights transferred to buyer
- No attribution required
- Commercial use allowed
- Requires written agreement

**TTS Service Licenses:**
- Varies by service (check terms)
- Usually allows commercial use with subscription
- May require attribution
- Check specific service terms

---

## Important Notes

1. **Always verify commercial use rights** before using any audio
2. **Save license documentation** - download license files or take screenshots
3. **Keep work-for-hire agreements** - essential for commissioned work
4. **Document everything** - better to over-document than under-document
5. **Voice lines must be parody** - never use actual Trump recordings
6. **Update this file immediately** when adding new audio assets

---

## Questions or Issues

If you're unsure about licensing:
1. Check the license terms on the source website
2. Contact the author/creator for clarification
3. When in doubt, choose a different source with clear licensing
4. Consider commissioning custom work with work-for-hire agreement
